# DanaApp

A simple cross-platform food donation application built using **Flutter** (frontend) and **Django REST Framework** (backend).  
The app connects donors, NGOs, and recipients to reduce food waste and streamline the donation process.

---

## Features
- User authentication (Firebase)
- Create and view food donations
- Upload and display food images
- Location-based pickup points
- NGO verification
- REST API with Django + PostgreSQL
- Flutter mobile UI with API integration

---

## Tech Stack
### Frontend
- Flutter
- Firebase Auth & FCM
- Geolocator

### Backend
- Django REST Framework
- PostgreSQL
- Media file support for image uploads
